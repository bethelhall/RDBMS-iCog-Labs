## Relational Database Using Basic Set Theory

### Functions implemented

The program implements the following Relational Algebra functions:

1. PROJECT

2. UNION

3. SET DIFFERENCE


#### Details of Operations

##### 1. `UNION` Operation:
```
UNION TableA TableB
```
##### 2. `DIFERENCE` Operation:
```
DIFF TableA TableB
```
##### 3. `PROJECT` Operation:
```
PROJECT Attributes TableA
```
